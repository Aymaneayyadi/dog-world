A modern, lightweight, SEO-optimized WordPress theme for dog websites.

== Installation ==

1. Upload the `dog-world-theme` folder to `/wp-content/themes/`
2. Activate the theme through the 'Appearance > Themes' menu in WordPress
3. Go to 'Appearance > Menus' to set up your navigation
4. Go to 'Appearance > Widgets' to add widgets to sidebar and footer areas
5. Create pages: About, Contact, Privacy Policy, Terms of Service
   - Assign templates: About Page, Contact Page, Privacy Policy Page, Terms of Service Page
6. Create categories: Dog Health, Dog Food, Dog Training, Puppy Care, Dog Breeds, Dog Behavior, Grooming, Dog Accessories
7. Set your front page: Settings > Reading > "Your latest posts"

== Features ==
- Modern responsive design
- SEO optimized with schema markup
- Google Core Web Vitals ready
- Ad placement widgets (before content, after content, in content)
- Custom logo support
- Primary and footer menu locations
- 4 footer widget areas
- Contact form built-in (shortcode: [dogworld_contact_form])
- Category icons on homepage
- Related posts
- Breadcrumbs
- Search bar in header
- Mobile-first design
- Fast loading (minimal dependencies)

== Ad Monetization ==
The theme includes dedicated widget areas for ads:
- Ad Before Content
- Ad After Content  
- Ad In Content
- Sidebar

Simply add your ad code (Google AdSense, Media.net, Monum, etc.) as HTML widgets.
